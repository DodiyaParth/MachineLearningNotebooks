    AppliedAI 

    By seeing observations from the following link, it should be abundantly clear that setting aside variables known to be strongly correlated with others can have a substantial effect on the PCA results because more correlated variables will over-emphasize particular eigenvectors, (directions), and if there are many correlated variables, then there would be so many more overemphasized 'fake' directions, that drown out an 'original' eigenvector/direction that would have otherwise been easily seen. So you can use correlation co-efficients to determine which variables are highly correlated . A cut off for highly correlated is often 0.8.

    Nov 06, 2018 10:16 AM

    Applied AI Course Team1 

    The cost function of t-sne is an alternative of the cost function of SNE. We can say that t-SNE cost function is a symmetric version of SNE. Because In t-sne, we try to minimize a single kl-divergence between the joint probability distribution P in higher dimensional space and a joint probability distribution Q in the lower dimensional space. But in SNE, we will try to minimize the SUM of kl-divergences between the conditional probabilities. In t-sne, the cost function is symmetric because it has a property that probabilities are equal. We can easily get that from the formula. For mathematical formulation and further details, please refer to this documentation.
